# 🌙 Noor-Ramadan

## 📖 Overview

**Noor-Ramadan** is a comprehensive Islamic lifestyle management system designed to enrich your spiritual journey. Built with the modern Qt framework, this all-in-one platform integrates essential Islamic tools with smart tracking and gamification to help you stay connected with your faith daily.

## ✨ Core Features

| Module | Description |
|--------|-------------|
| 🕌 **Prayer Management** | Accurate prayer times, Quran integration, and daily worship tracking |
| 📿 **Digital Tasbeeh** | Smart counter with daily goals, progress tracking, and multiple dhikr categories |
| 🏆 **Gamification Engine** | Reward points, achievement badges, and a global leaderboard |
| 🔊 **Adhan Service** | Real-time Adhan notifications with high-quality audio alerts |
| 📅 **Ramadan Calendar** | Interactive calendar with day tracking, duas, and special prayers |
| 📊 **Analytics Dashboard** | Visual spiritual progress reports and custom date-range analysis |
| 📚 **Educational Content** | Curated Islamic stories, daily hadith, and historical knowledge base |
| 👤 **User Management** | Secure profiles with customizable preferences and data backup |

## 🛠️ Technology Stack

- **Framework**: Qt 6.x (C++)
- **Database**: SQLite for reliable local storage
- **Audio**: Qt Multimedia for Adhan playback
- **UI**: Qt Quick/QML with modern, responsive design
- **Platform Support**: Windows, macOS, Linux

## 🚀 Getting Started

### Prerequisites
- Qt 6.x development environment
- C++ compiler (MinGW/GCC/MSVC)
- CMake 3.16+

### Installation

```bash
# Clone the repository
git clone https://github.com/abdosabra2007-ux/Noor-Ramadan-.git

# Navigate to project directory
cd Noor-Ramadan-

# Build with CMake
mkdir build && cd build
cmake ..
make

# Run the application
./NoorRamadan
```
